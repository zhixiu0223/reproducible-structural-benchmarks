# Frame 1 -- source data pointer

This repo does not redistribute the original dataset. Download directly
from the source (CC BY 4.0):

Ziemian, C.W., Ziemian, R.D. (2021). Benchmark Frames for Structural
Analysis. Mendeley Data, v1. https://doi.org/10.17632/39sjhchwtx.1

Files used for Frame 1:
- `Description Of Frames.pdf`, page 3 (Frame 1: Single story portal frame)
- `FEA Model Details_All frames.xlsx`, sheet "Frame 1"
- `Simulation Results_All frames-dYSR8w.xlsx`, sheet "Frame 1" (WC/SIPC/LA
  benchmark values used for comparison)

Companion methodology paper (source of the 0.8E Direct Analysis Method
stiffness requirement, not included in the Data in Brief release itself):
C.W. Ziemian, R.D. Ziemian (2021). Efficient geometric nonlinear elastic
analysis for design of steel structures: Benchmark studies. Journal of
Constructional Steel Research, 186, 106870.
